# Benchmark Results

Generated benchmark artifacts are written here by:

- `python3 scripts/run_benchmark.py`
- `python3 scripts/competition_mode.py`
- `POST /benchmark/export`

The `latest/` directory is refreshed on every export and contains:

- `benchmark-report.json`
- `stress-report.json`
- `page-results.csv`
- `action-results.csv`
- `stress-cases.csv`
- `SUMMARY.md`
- `COMPETITION_MODE.md` when competition mode is run
